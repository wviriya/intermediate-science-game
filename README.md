# Science Mastery - Year 8 Term 2

An interactive, gamified learning app for Year 8 Science focusing on Term 2 topics.

## Features

- 📱 iPhone-optimized responsive design
- 🎮 Gamification with points and achievement badges
- 📚 13 weeks of curated curriculum
- 🧠 Interactive quizzes (2 per week)
- 📊 Progress tracking
- 🎨 Visual diagrams and key points
- 📝 Based on actual exam questions

## Topics Covered

- Week 1-3: Respiration & the Respiratory System
- Week 4-6: Forces and Motion
- Week 7-8: Magnetism
- Week 9-10: Diet & Growth / Skeleton & Muscles
- Week 11-12: Ecosystems & Adaptations
- Week 13: Properties of Materials

## Installation

```bash
npm install
```

## Running Locally

```bash
npm start
```

The app will open at `http://localhost:3000`

## Deployment

This app is deployed on Vercel. See deployment instructions below.

## Tech Stack

- React 18
- Tailwind CSS
- Lucide React Icons

## How to Use

1. Select a week from the curriculum
2. Read the key points
3. Study the visual diagram
4. Answer the challenge question
5. Take the quiz to test understanding
6. Track your progress with achievements

## Learning Tips

- 30 minutes per week per topic
- Focus on understanding, not memorizing
- Explain concepts aloud to someone
- Review weak areas before exams
- Complete all quiz questions
